    • Research Experience

  • Fault diagnosis and prognosis in multivariate complex systems

    Combining the empirical probability density estimation technique with the cumulative sum approach for incipient fault detection
    Proposing a new one-class classification approach that determines a healthy region with multiple centers and a region radius
    ProposingtheLocalMahalanobisDistance(LMD)andtherobustanchor-generation algorithm to implement the healthy region modeling
    Developing data-driven methodologies based on the LMD index for three fault diagnosis tasks: fault detection, fault root cause isolation, and fault severity assessment.

  • Open circuit fault diagnosis for interleaved DC-DC converters

    Establishing the system mode of multi-phase interleaved DC-DC converters and estimating the state variable as the signature for fault detection
    Building the simulation environment and the experimental set-up for validating the effectiveness of the proposed method

  • Receiver Operation Characteristic (ROC) analysis and its applications

    Proposing a fast algorithm for estimating the volume under the ROC surface
    Applying the ROC analysis to the change point detection problem
    Developing Python packages and software for multi-class ROC analysis
